## Environment construction method
Homebrew install

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

pip install
```
brew install python3
```

Additional library install
```
pip3 install requests
pip3 install bs4
pip3 install json
pip3 install numpy
```
